# WHU-ComputerOrganizationDesign-lab
## 概述
武汉大学计算机学院 计算机组成原理课程设计 riscv版<br>
从2023 spring 开始，这门课程要求使用Verilog实现三个包括riscv指令集中部分指令的CPU。这三个CPU是单周期CPU，不带冒险检测的流水线CPU和带冒险检测与处理的流水线CPU。<br>
